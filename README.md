# Member Portal

A React + TypeScript Vite application for Church of Pentecost UK members to sign in and review their personal membership details.

## Getting started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Configure the API base URL (already provided) in `.env`:
   ```bash
   VITE_API_BASE_URL=https://testcopukservices.azurewebsites.net
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

## Available scripts
- `npm run dev` – start the Vite dev server
- `npm run build` – type-check and build for production
- `npm run preview` – preview the production build
- `npm run lint` – run ESLint on the codebase
